Docker Machine

Som vi så i forrige oppgave er docker-apper fine og portable, men å lagre til fil og flytte de rundt manuelt er ikke hverken smidig eller praktisk, og skalerer dårlig til store systemer. Docker-machine kan brukes for å automatisere dette. Docker-machine støtter de fleste store hosting providers, og man trenger faktisk kun EN kommando i docker-machine for å automatisk starte opp en server i skyen, installere og konfigurere docker på serveren og sette opp SSH med keypair!

a)

Bruk dockers create-kommando for å starte opp en Amazon AWS-server med utdelt API-key. Last deretter opp imaget laget i forrige oppgave.

b)

Bruk docker-machines innebygde ssh-kommando til å logge inn på serveren. Her kan man bruke samme docker-kommandoer som lokalt, for å verifisere at docker-containeren kjører!
